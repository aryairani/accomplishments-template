# USG accomplishments template

to build:
```
$ sbt one-jar
[info] Packaging blah/accomplishments_2.10-0.1-SNAPSHOT-one-jar.jar ...
```

to run:
```
$ java -jar blah/accomplishments_2.10-0.1-SNAPSHOT-one-jar.jar < my-tab-separated-csv.csv > output.txt
```
or
```
$ java -jar blah/accomplishments_2.10-0.1-SNAPSHOT-one-jar.jar my-tab-separated-csv.csv > output.txt
```
